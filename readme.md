### Nintendo Switchハード販売における主力タイトルの寄与分析 - 時系列回帰・安定性選択・残差分析による定量評価

任天堂の決算資料から取得したデータを用いて、Nintendo Switch本体販売台数と自社タイトル販売本数との関係を分析。
ハード販売に対して各タイトルがどの程度寄与しているか、その影響の強さと持続性を評価した。

[分析レポート(pdf)へのリンク](./00_report.pdf)

---

### 開発環境

- **Python 3.11.11**
- パッケージ管理：**pip**
- 分析環境：**Jupyter Notebook**

---

### ファイル／ディレクトリ構成

- **`00_report.pdf`**  
  分析のレポート

- **`01_analyze_switch.ipynb`**  
  分析ノートブック  
  `completed_deta/` に格納した前処理済みデータを回帰分析、安定性選択、スライディングウィンドウ回帰などを実行

- **`01_download_materials.ipynb`**  
  任天堂サイトの決算資料から販売データをダウンロードするためのコード

- **`completed_deta/`**  
  分析に使用するための前処理済みCSVデータ一式を格納
  - タイトル販売（累積／四半期別）  
  - Switchハード販売  
  - 四半期ダミー  
などを結合したデータ

- **`titles_sales_data/`**  
  任天堂決算資料から抽出・整理したタイトル販売データと、中途生成ファイルを格納  
  - `normalized_titles_sales_data/`：標準化済みの販売データ  
  - `switch/`, `wii/`, `wiiu/`：各ハードの元データ及び整理済みデータ  
  - その他ファイル：`completed_deta/` を完成させるまでの中途生成ファイル
